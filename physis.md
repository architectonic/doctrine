# Physis

Physis names the system as a growing, adaptive organization rather than a static rule file.

A long-lived human-agent system should preserve identity by adapting responsibly, not by freezing itself against change.

## Principle

```text
Static systems preserve identity by resisting change.
Living systems preserve identity by metabolizing change.
```

In Architectonic terms, Physis connects:

```text
constitution   what must remain true
knowledge      what changes as evidence changes
skills         what improves as craft improves
meta           how upkeep detects drift and revises the system
```

## Guardrail

Adaptive change is not uncontrolled mutation.

Every durable change should remain governed by doctrine, source status, authority, and audit.
